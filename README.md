# Smart EV Infrastructure Analysis

## Overview

This project focuses on analyzing Electric Vehicle (EV) charging data using SQL and Power BI.
It provides actionable insights into charging patterns, station performance, revenue trends, and energy consumption.

### Data Cleaning & Preprocessing
- Cleaned and structured raw EV charging datasets  
- Handled missing values and inconsistencies  
- Converted and formatted datetime fields 
- Removed duplicate records and treated outliers  
- Transformed data into analysis-ready format for SQL & Power BI

## Business Intelligence Analysis using SQL
Leveraged SQL to analyze EV charging data across stations and cities. Applied aggregation, joins, and window functions to uncover key trends. Provided actionable insights for business optimization.

The SQL operations are performed using Pandas in a Jupyter Notebook (.ipynb) for efficient data analysis.<br>
